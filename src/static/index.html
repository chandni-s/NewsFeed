Add Keyword

A keyword is a word to look for when searching through articles. It may be associated with a source, so that a reference is automatically created when it is seen.

Add URL

A watched URL will be checked automatically for new articles. A watched URL can be the front page of website, a page on a website for breaking news, a twitter account, or an RSS feed.

Add article

An article can either be added from a URL, or entered in manually through the article editor. If added from URL, you will be taken to the results page.

From URL

If checked, URLs in the given URL will be parsed too. This may create articles which are not published pieces of news, depending on the site. If unsure, leave unchecked.
